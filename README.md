# 令和8年度E3B１班の課題研究

## 課題研究テーマ
Bluetoothを使った自動資料読込み機

## 6月12日までの研究の進捗
Arduino UNO R4 本体とPCとを接続し、Wi-Fi連携機能を使ってスマートフォンからの遠隔操作をできるようにした。

Arduino UNO R4 とブレッドボードを接続させブレッドボードにあるLEDをスマートフォンで遠隔でON・OFF できるようになった。

![プログラムコード例](https://github.com/Heiya3/R8.kadai.No1/assets/240632538/image2.jpg)

## 現段階での課題点
- Arduino UNO R4 とブレッドボードを接続し、スマートフォンから遠隔でブレッドボードのLEDを光らせることができた
- Wi-Fi連携により遠隔操作が可能になった
